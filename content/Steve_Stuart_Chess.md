**Steve Stuart Chess**,  

an early [microprocessor](https://en.wikipedia.org/wiki/Microprocessor) chess program written in 1977/78 by [Stephen Stuart](Stephen_Stuart "Stephen Stuart") in [Assembly](Assembly "Assembly") for a home-made metal-box computer system <a id="cite-note-1" href="#cite-ref-1">[1]</a> with a [Signetics 2650](https://en.wikipedia.org/wiki/Signetics_2650) processor. Stuart was challenged for a microcomputer chess program by an [Amdahl](https://en.wikipedia.org/wiki/Amdahl_Corporation) engineer to the Amdahl Computer Club, where he was member <a id="cite-note-2" href="#cite-ref-2">[2]</a>. Steve Stuart Chess played the [Second West Coast Computer Faire Microcomputer Chess Tournament](MCCT_1978 "MCCT 1978") in March 1978 in the "less than 8K of memory class b" <a id="cite-note-3" href="#cite-ref-3">[3]</a>.
